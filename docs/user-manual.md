# User Manual

## Open Documents

1. Click `Browse` or enter an absolute path in the inspector.
2. Open additional PDFs to add more tabs to the workspace.
3. Reopening an already-open file switches back to the existing tab.

## Reorder And Edit Pages

- Drag pages inside the main grid to reorder them.
- Use the page toolbar or context menu to rotate, duplicate, delete, or insert blank pages.
- `Undo` / `Redo` works per session and tracks in-place operations through snapshot history.

## Work With Multiple Documents

- Enable `Split view` to compare two open PDFs side by side.
- Drag a page from the primary document into the secondary document to move it there.
- Use `Merge selected` to combine selected tabs into a new PDF.

## Crop Pages

1. Select one or more pages.
2. Click `Crop`.
3. Adjust margins with drag handles or numeric inputs.
4. Optionally enable aspect-ratio lock and preview.
5. Click `Apply crop` to update all selected pages.

## Import Images

1. Choose the insert mode: `After selection`, `Append`, or `Prepend`.
2. Click `Import images`.
3. Select one or more supported images.
4. The imported pages are inserted in the selected order and become the active selection.

Supported formats:

- JPEG / JPG
- PNG
- TIFF / TIF
- BMP
- WebP

## Mobile-Oriented Use

- On narrow screens, the workspace shows a fixed bottom toolbar with import, crop, rotate, undo, and delete actions.
- Pinch inside the page grid to zoom the thumbnail density in or out.
- Device-profile smoke tests currently cover desktop, iPhone 13 viewport, and Pixel 7 viewport.
